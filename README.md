# Quickstart

First, install PDM. 
```
# Instructions for Linux
curl -sSL https://pdm-project.org/install-pdm.py | python3 -
echo "export PATH=~/.local/bin:$PATH" >> ~/.bashrc
source ~/.bashrc
```

Now, install the repository. 
```bash
pdm install
```
